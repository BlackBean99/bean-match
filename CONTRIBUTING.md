# CONTRIBUTING.md

## Workflow
1. Read the core docs.
2. Create a scoped branch.
3. Follow RPI: Read -> Plan -> Implement.
4. Validate locally.
5. Open PR with the template.
6. Merge only after required checks pass.

## Commit Convention
Use Conventional Commits:
- feat(scope): summary
- fix(scope): summary
- docs(scope): summary

## Definition of Done
See `docs/operations/11-Code-Complete-and-Definition-of-Done.md`.
