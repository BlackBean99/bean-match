# Human Approval Checklist

- [ ] Scope matches the original request
- [ ] Acceptance criteria are explicitly met
- [ ] Lint, typecheck, build, and required tests are attached
- [ ] Security or privacy review is attached when applicable
- [ ] Code review findings are resolved or accepted
- [ ] Rollback path is documented
- [ ] No unauthorized schema, auth, privacy, infra, or deletion changes are included
- [ ] Merge target and branch strategy follow repo policy
